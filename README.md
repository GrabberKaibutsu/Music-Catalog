# SoundScope

An inventory management app for personal or professtional use. Can track the name, count, and if it needs to be ordered or restocked.

## Technologies Used


1. CSS 
2. HTML 
3. Javascript 
4. Git Bash 
5. Visual Studio Code.
6. ChatGPT (for private sessions)
7. Dependencies
