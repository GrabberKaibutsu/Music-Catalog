# SoundScope

A music based social media application for keeping up with new relases and trending songs albums, and artists. Uses the Spotify API

## Technologies Used


1. CSS 
2. HTML 
3. Javascript 
4. Git Bash 
5. Visual Studio Code.
6. ChatGPT (for private sessions)
7. Dependencies
8. Spotify API
9. Vite
10. React
